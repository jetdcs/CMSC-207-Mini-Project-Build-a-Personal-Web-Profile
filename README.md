# CMSC 207 Mini-Project: Build a Personal Web Profile

Inspired by the website in the book *Create with Code: Build Your Own Website* by Clyde Hatter.

---

## Project Overview

This mini-project is a stepping stone towards the final project and focuses on applying foundational web development concepts. Students will create a basic personal web profile, following the principles taught in Clyde Hatter’s book.

The project will reinforce HTML structure, CSS styling, and basic JavaScript interactivity while keeping the scope manageable for mid-semester progress.

---

## Project Objectives

By completing this mini-project, students will:

- Develop a structured and semantic HTML page
- Apply CSS to enhance layout, colors, and typography
- Use JavaScript for simple interactivity
- Implement responsive design for mobile and desktop views
- Get comfortable with debugging and refining their web pages

---

## Project Requirements

### 1. Website Structure (Single Page Website)

The webpage should include the following sections:

- **Header** – Name, site title, and a simple navigation menu
- **About Me** – A short paragraph introducing the student with an image
- **Hobbies & Interests** – A list of at least three hobbies with brief descriptions
- **Contact Info** – A section displaying an email address and social media links

---

### 2. Technologies Used

- **HTML** – Use proper semantic tags (`<header>`, `<section>`, `<footer>`, etc.)
- **CSS** – External stylesheet for design and layout
- **JavaScript** – A simple script for interactivity (button click effects or changing content dynamically)
- **Optional Enhancements** – Basic CSS animations or transitions

---

## Features & Enhancements

- Navigation menu with working anchor links
- Hover effects for buttons or images
- JavaScript button that displays a fun fact about the student
- Mobile-friendly layout using CSS media queries
